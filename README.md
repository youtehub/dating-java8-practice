# Dating Java8系列视频

## 前言

**Dating Java8** 系列视频从实用性的角度出发介绍了Java8中的核心特性及其应用场景。

通过系列视频的学习后你将掌握较之以往更为巧妙的编码方式和编程技巧，从而提升我们的开发效率。

与Java8的**浪漫的相遇**将会使你忘掉过去繁冗的编码方式，只需用之前一半不到的时间就能写出更短、更清晰的代码。进阶到Java8之后你会意识到自己再也无法回到“旧Java时代”了。
